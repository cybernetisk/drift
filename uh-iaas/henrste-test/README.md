# uh-iaas for cyb

Dette er filer brukt for å sette opp prototype og leke med CoreOS
på UH-IaaS på UiO.

## Pakker på Ubuntu

```
python3-glanceclient
python3-openstackclient
python3-novaclient
```

## Ad-hoc kommandoer

http://docs.uh-iaas.no/en/latest/api.html

```
openstack server list

```
